Proxemics Personal Project

Description

We were tasked to use the idea of proxemics to make anything we wanted.

Some of the prompts given were:

  • A mobile interface for a smart object
  • A stationary display that solves a problem in your home
  • An application that helps support a multi-person in-person situation

Initial Sketch Description

I explored some ideas that were more utility based and some others that are using proxemics to enhance our experience

I personally felt that they ideas which were the most creative were the ones that enhanced the embodied experience of something already

These can be seen in Page 2 & Page 4

Page 2

This depicts 3 sketches which show different ways that proxemics could be used to produce music. THe first sketch is like twister, but the areas are covered with detectable symbols.

The second sketch shows someone using something that has detectable symbols which are mapped to musical instruments. The position and distance from the camera cause the sound to change.

The third sketch shows a similar idea to the above except that garbage bags are lined with these symbols which can then be wrapped around other objects, making anything a musical instrument.

Page 4

This shows 2 sketches which outline different ways a play/performance could be enhanced with the use of proxemics

The first sketch outlines how characters (which would be recognized somehow) could act out a performance and a narration could occur in tandem with their performance along with music. It could use AI or something to adapt to the script. It would use the position of the characters, their distance between each other, the distance from the camera, and other parameters to contribute to the performance.

The second sketch displays a similar idea except that the performance is done using puppets instead of actors. The difference being that one person could play multiple characters as well as the fact that it is harder to tell a story with puppets than it is from a live show.

Reflection

I personally think that I may have gone a bit overboard with the implementation as it took way too much time than it needed.

Overall, I'm pretty satisfied with the way it came out, though I wanted to add some more features, but thats for another day.

I think I liked this assignment the most as this was the one where I had the most creative ideas I think and so it was pretty hard for me to pick what idea to continue with. Specifically, I really like my play/puppet show idea and thought it had potential however I thought that it might have been too much.

Also, I wish I had some teammates because I think some of my ideas need to be tempered and processed by others so that they can help me filter the good and bad ideas.

I had a lot of fun making this project and I think I showed to myself the variety of ideas that I can produce, I just need to learn on filtering them appropriately.
